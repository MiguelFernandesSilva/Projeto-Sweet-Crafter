## Sistema de Encomendas Customizaveis

Um ecommerce de doces onde o usuário pode fazer encomendas customizaveis e entrar em contato com confeiteiros locais para realizar a compra

## Tecnologias Atuais 

- HTML
- CSS
- JavaScript

## Tecnologias Futuras

- Banco de Dados (PostgreeSQL)
- Nuvem (AWS)
- Backend (Node.js)
- Framework (React)

## Demonstração 

Acesse os arquivos em: https://github.com/MiguelFernandesSilva/Projeto-Sweet-Crafter

## Aprendizado / Melhorias 

- Melhoria na Estruturação e Planejamento de Códigos
- Aprofundamento em Tecnologias do CSS, Tais como Pseudoclasses, Pseudoelementos, Variáveis e Outros
- Aprofundamento no Uso de Funções e Métodos
- Aprendizado em Armazenamento de Dados no Navegador e Manipulação destes Dados

## Funcionalidades

- Demonstração Detalhada Automática de Produtos
- Lógico Seguro 
- Cadastro de Usuários 

## Funcionalidades Futuras

- "Favoritar" Produtos

- Armazenamento Seguro de Dados em Banco de Dados
